pycharm快捷键信息：
Ctrl + Alt + Space: 代码提示
Ctrl + P: 参数信息提示（在方法括号内使用）
Ctrl + Q: 快速查看文档，在方法名后使用
Ctrl + Z: 后退
Ctrl + Shift + Z: 前进
Ctrl + / 行注释/取消注释
Ctrl + Shift + / 块注释
Ctrl + Alt + O: 优化导入，去掉一些没用的导入
Ctrl + Alt + L: 代码格式化
Ctrl + D: 复制选定的区域或行
Ctrl + Y: 删除选定的行
Ctrl + Delete|Backspace: 一次删除一个单词或一个符号
