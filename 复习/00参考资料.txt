pycharm快捷键信息：
Ctrl + Alt + Space: 代码提示
Ctrl + P: 参数信息提示（在方法括号内使用）
Ctrl + Q: 快速查看文档，在方法名后使用
Ctrl + Z: 后退
Ctrl + Shift + Z: 前进
Ctrl + / 行注释/取消注释
Ctrl + Shift + / 块注释
Ctrl + Alt + O: 优化导入，去掉一些没用的导入
Ctrl + Alt + L: 代码格式化
Ctrl + D: 复制选定的区域或行
Ctrl + Y: 删除选定的行
Ctrl + Delete|Backspace: 一次删除一个单词或一个符号

1、为什么python被称为胶水语言？他是新一代的系统脚本
参考博客：https://www.cnblogs.com/ningskyer/articles/5264172.html

2、python百度百科介绍：最初被设计用于编写自动化脚本(shell)
可以用于web和internet开发、科学计算和统计、桌面界面开发、软件开发、后端开发

3、检查是否安装了python：在命令行中输入python,如果有响应表示安装了python的，会在响应中显示python版本信息

4、十分经典的科学计算扩展库：numpy,scipy,matplotlib分别提供了数组处理、数值运算、绘图功能

5、python是一种限制性很强的语法，不好的编程习惯不能通过编译，很重要的一项就是python的缩进规则

6、python可以进行多继承、重载运算符

7、脚本语言一般是指作简单程序设计任务的语言，只能处理简单任务的编程语言

8、Python的执行，首先会将.py文件中的源代码编译成Python的字节码，再由Python虚拟机来执行这些编译好的字节码。
这种机制的基本思想跟Java是一致的。然而，python虚拟机与Java虚拟机不同的是，python虚拟机的抽象层次更高。

9、python的缩进：必须使用4个空格来表示每级缩进，使用table缩进也可以编译通过

10、python采用动态类型系统，在编译的时候，python不会检查对象是否拥有被调用的方法或属性，直至运行时，才做出检查，所以操作对象可能会抛出异常。
不过，python禁止没有明确定义的操作，比如数字追加字符串。
